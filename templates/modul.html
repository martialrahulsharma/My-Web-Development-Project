How to install module

BACK
NEXT

To install module in python using pip is package manager of python.

syntax

pip install modulename

Example

pip install flask

There are two types of python module.

  1. Built in module:

    2. Already installed in system during installing python.
    Example:- random, math, etc.

  2. External module:

    3. External module is not installed during python. Using internet for installing external module.
    Example: pandas, flask, flask_sqlalchemy, flask_login, etc.

Don't create file name to module name

Interpreter of python first search in same folder where your file is created and after in python folder like bin folder.
Example
flask.py (don't choose)
hello.py (choose)
flask is module of python.
