How to create list in python ?

BACK
NEXT

List in python is be in square brackets and value in inverted commas.

syntax

list = ["value1","value2","value n"]

Example

list = ["python","flask","ajax","HTML","CSS"]
print(list)
Output:
['python', 'flask', 'ajax','HTML','CSS']

List start with 0 index.

Example: 
list = ["python","flask","ajax","HTML","CSS"]
print(list[0],list[1],list[2])
Output:
python flask ajax

List of integer

list of integer is not be in inverted commas.

Example: 
list = [1,2,3,4]
print(list)
Output:
[1,2,3,4]

How to access of index of list ?

list = [1,2,3,4]
print(list[1])
Output:
2

Why: because index is start from 0 (zero).

List function

sort()

sort() method is used to sort the list.

Example: 
list = [5,2,4,1,3]
list.sort()    
print(list)
Output:
[1, 2, 3, 4, 5]

reverse()

reverse() method is used to print the list in reverse order.

Example: 
list = [5,1,3,6,8,3,5]
list.reverse()    
print(list)
Output:
[5,3,8,6,3,1,5]

List slicing

It is working like string slicing, lets see an example.

Example: 
list = [5,5,2,3,4,1,6]
print(list[0:3])
print(list[0:6:2])
Output:
[5, 5, 2]
[5, 2, 4]

Why: because first output start from 0 index to 2 index and second output start from 0 index to 5 index and skip 1, 3, 5,...index. Negative slicing not used in list.

max() and min()

max() method return maximum value from list and min() method return minimum value from list.

Example: 
list = [5,5,2,3,4,1,6]
print("min value from list: ",min(list))
print("max value from list: ",max(list))
Output:
min value from list: 1
max value from list: 6
