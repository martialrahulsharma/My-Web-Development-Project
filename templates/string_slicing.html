String slicing and other function

BACK

mystr = "I am Yamraj"
print(mystr[0:5])
Output
I am
above example output 0 to 4 string will be print because 5 will be exclude, it is called string slicing.
if we will give extra range in mystr[75] then a error occured that error will be: 'out of range'. 
print(mystr[75])
IndexError: string index out of range 
print(mystr[0:75])
no any error:
I am Yamraj
if want to print skip characters then: 

print(mystr[0:10:2])
third index nmber in mystr[0:10:2] is 2 i.e it is skip characters between mystr.
if second index not define in mystr then by default will be length of string mystr. Example 
print(mystr[0:])
Output
I am Yamraj
if first, second and third index value not given then by default first index will be 0, second length of string and third will be 1. 
print(mystr[ : : ])
Output
I am Yamraj 
print(mystr[0: : ])
Output
I am Yamraj
Why: mystr[0 : length of string : default 1]
print(mystr[0:10: ])
Output
I am Yamraj
Why: mystr[0 : 10 : default 1]
print(mystr[ :10: ])
Output
I am Yamraj
Why: mystr[default 0 : 10 : default 1]


If want to print apposit side string then use bellow code: 

print(mystr[ : : -1])
Output
jarmaY ma I

Other functions

isalnum()

isalnum() function return boolean character True or False. Alphanumeric characters are be without space.
Example: 

mystr = "I am Yamraj"
print(mystr.isalnum())
Output
False
Example: without space 
mystr = "IamYamraj"
print(mystr.isalnum())
Output
True

isalpha()

isalnum() function return boolean character True or False. It will be also same output .
Example: 

mystr = "I am Yamraj"
print(mystr.isalpha())
Output
False
Example: without space 
mystr = "IamYamraj"
print(mystr.isalpha())
Output
True
