There are two types of comment in python.

  1. Single line comment
  2. Multi line comment

Single line comment

Using # for single line comment.
Example:

# this is comment of python for single line

Multi line comment

Using tripple inverted commas for Multi line comment.
Example

'''Hi i am 
multiline
comment'''
or
"""Hi i am 
also
multi line comment"""

If following code will run then output will come in two line. 

print("India is Great")
print("There are many rivers")
but if we add end=" " statement in print then new line will not be print. 
print('My name is', end=" ")
print("Yamraj")
if add somthing in end="somthing" then somthing data will be print added. 
print('My name is', end=" and ")
print("Yamraj")

Escape Characters

There are following escape characters in python.

  • \' (Single Quote)
  • \\ (Backslash)
  • \n (New Line)
  • \r (Carriage Return)
  • \t (Tab)
  • \b (Backspace)
  • \f (Form Feed)
  • \ooo (Octal value)
  • \xhh (Hex value)
see link more info: https://www.w3schools.com/python/gloss_python_escape_characters.asp
